# KyraGameEngine
KyraGameEngine is a compact 2d/3d-game engine for low-latency, multithreaded game development
providing a lightweight renderer with a configurable RenderPipeline and render-pass system, windowing and input, a scene graph with 2D support, a scriptable component system and audio playback.

The repo includes runnable example (Pong)
